1 Bienvenue & Introduction

L'université est un lieu d'apprentissage, d'épanouissement et d'échanges. Ce module est consacré aux nouveaux inscrits pour préparer leur cycle de licence.
Exemple de Mehdi : Nouveau bachelier, il se pose des questions sur l'adaptation, la gestion du temps et les ressources pour réussir. Ce module vise à lui fournir les outils nécessaires.

2 Le Système d'Enseignement

Au Maroc, le système comprend trois secteurs :

  • Public : Universités et établissements sous tutelle ministérielle.
  • Privé : Créé par initiatives privées.
  • Partenariat : Fondations à but non lucratif.

Accès (Dakhla)

1. Accès Ouvert : Pour tous les bacheliers (ex: Facultés).

2. Accès Régulé : Admission limitée sur étude de dossier ou concours (ex: Écoles, Instituts).

3 Système LMD (Licence - Master - Doctorat)

  • Cycle : Étapes successives (L, M, D).
  • Filière : Cursus de formation.
  • Module : Unité fondamentale (Matière) de ~50h.

Validation

Validation Module ≥ 10/20. Validation Semestre par compensation si Moyenne ≥ 10/20 (aucune note < 5/20).
